# ceramic-dreams-online-store
This is an open-source, Vue.js-based online store application that comes with built-in authentication features, allowing users to securely sign up, log in, and manage their accounts. 

## Features
User Authentication: Our application includes a robust authentication system that ensures the security and privacy of user data. Users can create accounts, log in, and recover forgotten passwords through a secure and user-friendly interface.

# Product Catalog: 
Showcase your products in a beautiful and organized catalog, complete with product details, pricing, and images. Users can easily browse through the catalog and add items to their shopping carts.

# Shopping Cart: 
The application comes with a  shopping cart, allowing users to manage their selected items, review their orders, and proceed to checkout.

# Order Management: 
Administrators have access to an intuitive dashboard where they can manage and track incoming orders, update order statuses, and handle order processing efficiently.

# Responsive Design: 
Our online store application is built with responsiveness in mind, ensuring that customers can access and use the store seamlessly across various devices, including desktops, tablets, and smartphones.

# Project setup
### Installation
To get started with the application, follow these simple steps:

1- Clone the repository to your local machine using the following command:
```
https://github.com/maryamfad/ceramic-dreams-online-store.git
```
2- Change into the project directory:
```
cd ceramic-dreams-online-store
```
3- Install the required dependencies using npm
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
